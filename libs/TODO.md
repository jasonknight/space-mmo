# List of Major Components that Must Exist

* i18n for translations of the interface and logging
* Logging - how do we create structured logs for the game?
* Game client settings, or CLI settings
* Basic UI library (Text input, Slider, Toggle, TextDisplay(bold, italic), Button(hover/click))
* Item System and Database
* Inventory System 
* Chat System
* Console interface to change settings, levels, trigger actions etc.
